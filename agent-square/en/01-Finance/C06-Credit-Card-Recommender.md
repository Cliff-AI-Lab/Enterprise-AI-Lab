# 💳 信用卡推荐引擎

## Use Case
比价平台/银行App给用户推荐最合适的信用卡产品。

## Core Capabilities
- 消费画像分析
- 权益-消费匹配
- 积分/返现最优
- 里程兑换策略

## Sample Output
```
【用户画像匹配】
用户: 月消费¥12,000, 常出差, 海外购物30%
推荐组合:
  1. 招行Visa白金 (海外返现3% + 机场贵宾)
  2. 建行JAL联名卡 (日本消费2x里程)
  3. 运通白金 (酒店+餐饮权益)
预估年值: ¥4,200 (返现+积分兑换)
```
