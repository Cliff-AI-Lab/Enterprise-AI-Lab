# 🎯 众筹平台运营 Crowdfunding Platform Ops

## Use Case
Kickstarter/摩点/小红花众筹平台: 项目审核+社交传播+支付+履约。

## Core Capabilities
- 项目审核+立项
- 达成率预测
- 支持者运营
- 履约跟踪

## Bound APIs
| API | Purpose |
|-----|------|
| Stripe | 支付+退款 |
| Instagram + Reddit | 社交传播 |
| Shopify | 商品上线 |
