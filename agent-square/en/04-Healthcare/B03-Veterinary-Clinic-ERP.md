# 🐾 宠物医院管理 Veterinary Clinic ERP

## Use Case
连锁宠物医院的HIS+客户+药品+用品管理。

## Core Capabilities
- 电子病历
- 排班+挂号
- 药品+耗材
- 会员回访

## Bound APIs (Animals类别)
| API | Purpose |
|-----|------|
| OpenFDA Animal | 宠物药品 |
| Dog API / Cat API | 品种数据库 |
| Shopify | 周边用品销售 |
