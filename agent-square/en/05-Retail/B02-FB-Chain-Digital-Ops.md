# 🍜 餐饮连锁数智 F&B Chain Digital Ops

## Use Case
餐饮连锁品牌(茶饮/快餐/正餐): 中央厨房+外卖+堂食+会员+供应链一体化。

## Agent Composition
```
[Hospitality Guest Services] ← specialized
[Private Domain Operator] ← marketing
[新零售门店运营] ← Growth Hacker
[供应链策略师] ← Supply Chain Strategist
```

## Bound APIs
| API | Purpose |
|-----|------|
| OpenFoodFacts + Edamam | 营养成分/食谱 |
| AfterShip | 食材物流 |
| WeChat Open | 会员小程序 |

## 核心工作流
1. **中央厨房**：SOP+半成品
2. **门店配送**：每日/每周
3. **点餐数字化**：堂食扫码+外卖
4. **会员运营**：储值+券+打卡
5. **数据洞察**：坪效+复购
