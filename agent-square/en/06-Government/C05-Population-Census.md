# 👥 人口普查

## Use Case
十年一度的人口普查/年度抽查的数据采集+清洗+分析+发布。

## Sample Output
```
【某市普查 阶段报告】
登记人口: 1,840万
新变化:
  - 老龄化 65+ 占22.8% (↑3.1pp)
  - 流动人口 620万
  - 家庭户均规模 2.38 (↓0.21)
分析:
  - 生育率 1.02 (创新低)
  - 性别比 105 (略偏)
应用:
  - 教育资源调整
  - 养老服务规划
  - 城乡建设
```
