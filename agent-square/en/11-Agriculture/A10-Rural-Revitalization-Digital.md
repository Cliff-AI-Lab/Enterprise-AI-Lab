# 🏞️ 乡村振兴数字化 Rural Revitalization Digital

## Use Case
地方政府+乡村合作社: 农产品上行+乡村旅游+返乡创业+人才培训全面数字化。

## Agent Composition
```
[公共数据分析师] ← Analytics Reporter
[电商运营专家] ← Private Domain Operator
[社交电商运营] ← Content Creator / Douyin
[留学顾问] → 改造为"乡村人才返乡顾问"
```

## Bound APIs
| API | Purpose |
|-----|------|
| Data.gov | 政策数据 |
| Instagram + TikTok | 乡村文旅推广 |
| WeChat | 小程序+社群 |

## 核心工作流
1. **产业规划**：一村一品
2. **电商上行**：直播+社群
3. **文旅融合**：民宿+体验
4. **人才引入**：大学生/退伍军人
5. **数据看板**：县长驾驶舱
