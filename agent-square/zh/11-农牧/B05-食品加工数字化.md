# 🏭 食品加工数字化

## 应用场景
中央厨房/速冻食品/调味品厂的HACCP+追溯+品控数字化。

## 核心能力
- HACCP关键控制点
- 批次追溯
- AI视觉品检
- 配方管理

```
年产: 18万吨 | SKU 420
HACCP:
  - 关键点6个 全记录

品控:

  - 微生物检验 合格率 99.98%
```
