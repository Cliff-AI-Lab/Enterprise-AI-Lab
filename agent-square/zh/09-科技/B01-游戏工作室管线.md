# 🎮 游戏工作室管线 Game Studio Pipeline

## 应用场景
中大型游戏工作室全流程: 策划/程序/美术/音频/QA 多工种协同+CI/CD+版本管理。

```
[Game Designer] ← game-development

```

## 绑定API
| API | 用途 |
|-----|------|
| GitHub API | 代码+LFS |
| HuggingFace | AIGC素材生成 |
| Replicate | 模型托管 |
| Mixamo (via other) | 动画库 |

## 核心工作流
1. **立项**：GDD + 原型
2. **开发管线**：Perforce/Git LFS
3. **构建CI/CD**：每日构建+自动测试
4. **内容创作**：AI辅助美术/文案
5. **发布运营**：多平台签包+热更
