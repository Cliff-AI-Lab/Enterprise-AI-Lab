# 🎯 客户成功平台

## 应用场景
SaaS公司的客户健康度+NPS+流失预警+扩展。

## 核心能力
- 健康度评分
- 低活预警
- NPS回访
- 升级/交叉销售

## 典型输出
```
【CS平台 月报】
管理客户: 2,800家 | MRR $680K
健康度分布:
  - 绿色 (健康) 68%
  - 黄色 (关注) 22%
  - 红色 (危险) 10%
本月:
  - 流失挽回 22家 ARR $180K
  - 扩展销售 成交 $380K
  - NPS 48
```
