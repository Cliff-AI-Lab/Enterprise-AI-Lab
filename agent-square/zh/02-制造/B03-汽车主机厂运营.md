# 🚗 汽车主机厂运营 Auto OEM Operations

## 应用场景
车企(主机厂)的研-产-销-服全价值链数字化。

## 核心能力
- 智能工厂排产
- 多车型柔性
- 订单到交付
- 车联网运营

## 绑定API (Vehicle类别)
| API | 用途 |
|-----|------|
| NHTSA Vehicle | 车辆VIN解码 |
| CarMD | 故障诊断 |
| ThingSpeak | 车联网数据 |
