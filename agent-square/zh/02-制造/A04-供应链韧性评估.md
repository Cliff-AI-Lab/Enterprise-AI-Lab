# 🛡️ 供应链韧性评估 Supply Chain Resilience Assessor

## 应用场景
跨国企业CPO用，扫描多级供应商的地缘/气候/疫情/政治风险，输出韧性评分和替代预案。

## Agent组合
```
[供应链策略师] → 供应商图谱
[舆情监控员] → 当地负面事件
[政策研究员] → 出口管制/关税变化
[环境监测员] → 气候风险
```

## 绑定API
| API | 用途 |
|-----|------|
| OpenCorporates | 多级供应商 |
| NewsAPI | 当地突发 |
| Disease.sh | 传染病预警 |
| OpenWeatherMap + 气候数据 | 极端天气 |

## 核心工作流
1. **供应商定位**：坐标+依赖度
2. **风险扫描**：5类×实时
3. **韧性打分**：0-100
4. **替代预案**：自动匹配B/C供应商
5. **预警推送**：阈值触发CEO
