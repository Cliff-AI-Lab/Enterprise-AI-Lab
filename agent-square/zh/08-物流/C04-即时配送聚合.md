# ⚡ 即时配送聚合

## 应用场景
商家对接达达/蜂鸟/闪送的配送聚合平台, 自动选最优。

## 核心能力
- 多家比价
- 智能路由
- 异常兜底
- 统一对账

## 典型输出
```
【聚合平台 月度】
合作配送: 5家 (达达/蜂鸟/闪送/顺丰+美团)
商家: 28万家 | 月订单 8,200万
智能路由节省:
  - 平均运费↓12%
  - 时效↑8%
异常: 单平台故障自动切换
```
