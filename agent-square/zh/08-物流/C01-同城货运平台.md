# 🚐 同城货运平台

## 应用场景
货拉拉/滴滴货运: 企业/个人同城发货撮合。

## 核心能力
- 订单撮合
- 司机准入
- 保险+保价
- 多车型
- 司机端+客户端

## 典型输出
```
【同城货运 月度】
订单: 380万 | 司机 42万 (月活)
GMV: ¥6.8亿
车型:
  - 面包车 52%
  - 小货车 32%
  - 中型货车 16%
时效:
  - 接单 3分钟
  - 到达起点 15分钟
客诉: 0.35%
```
