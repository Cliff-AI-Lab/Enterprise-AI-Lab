# 🎯 消费金融风控 Consumer Finance Risk Control

## 应用场景
分期/信用卡/助贷平台的欺诈识别+资信评估+反撸口子。

## 核心能力
- 实时反欺诈评分
- 黑产设备指纹识别
- 薅羊毛聚集识别
- 多头借贷筛查

## 绑定API
| API | 用途 |
|-----|------|
| HIBP | 手机/邮箱泄露库 |
| OpenSanctions | 黑名单 |
| Alpha Vantage | 宏观利率 |
