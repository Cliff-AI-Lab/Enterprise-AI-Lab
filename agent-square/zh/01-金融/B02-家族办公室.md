# 👑 家族办公室 Family Office Manager

## 应用场景
超高净值家族(UHNW)的全球资产配置+税务+传承+家族治理。

```
[Chief of Staff] ← specialized
[投资组合管理师] ← Portfolio Manager
[税务筹划师] ← Tax Strategist

```

## 核心工作流
1. **资产盘点**：多国多币种
2. **配置策略**：8+大类
3. **税务筹划**：CRS/信托
4. **慈善+ESG**：影响力投资
5. **家族治理**：二代/三代培养
