# 🧑‍💻 职业培训闭环 Vocational Training Loop

## 应用场景
编程/设计/AI等新兴技能的全链路培训: 招生→教学→项目→就业推荐。

## Agent组合
```
[课程设计师] ← Corporate Training Designer
[AI模型评测师] ← Model QA Specialist
[销售数据分析师] ← Recruitment Specialist
```

## 绑定API
| API | 用途 |
|-----|------|
| GitHub API | 学员作品 |
| HuggingFace | 能力评估 |
| Remotive Jobs | 岗位对接 |

## 核心工作流
1. **招生筛选**：免费试听
2. **分班教学**：直播+录播
3. **项目实战**：开源作品
4. **能力认证**：考评体系
5. **就业推荐**：合作企业
