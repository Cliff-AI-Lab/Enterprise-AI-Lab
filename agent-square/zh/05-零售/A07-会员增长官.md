# 👥 会员增长官 Membership Growth Officer

## 应用场景
品牌会员体系设计+分层运营+复购拉升+流失预警全流程。

## Agent组合
```
[电商运营专家] ← Growth Hacker
[销售数据分析师] ← Sales Data Extraction
[客服智能助手] ← Customer Service
```

## 绑定API
| API | 用途 |
|-----|------|
| Shopify | 会员数据 |
| 邮件/SMS APIs | 触达 |
| QuickChart | 可视化 |

## 核心工作流
1. **分层体系**：RFM + LTV
2. **权益设计**：等级特权
3. **触达策略**：邮件/短信/私域
4. **流失预警**：模型预测
5. **ROI看板**：会员成本与收益
