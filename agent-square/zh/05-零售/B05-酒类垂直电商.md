# 🍷 酒类垂直电商 Wine & Spirits E-com

## 应用场景
葡萄酒/白酒/威士忌等酒类垂直电商: 选品+溯源+教育+配送。

## 核心能力
- 酒款百科+评分
- 年份+产地认证
- 品鉴会+教育
- 合规配送(酒类许可)

## 绑定API
| API | 用途 |
|-----|------|
| OpenFoodFacts | 成分/过敏原 |
| Shopify | 电商 |
| Instagram | 品牌传播 |
