# 🏪 新零售门店运营 New Retail Store Manager

## 应用场景
线下门店+线上小程序+私域社群一体化运营, 盒马/屈臣氏模式。

## Agent组合
```
[电商运营专家] ← Private Domain Operator (marketing)
[仓储物流协调员] → 门店库存+同城配送
[客服智能助手] ← Customer Service
```

## 绑定API
| API | 用途 |
|-----|------|
| Shopify / 自建 | 商城 |
| AfterShip | 同城配送 |
| WeChat Open | 小程序+企微 |

## 核心工作流
1. **会员中台**：线上线下统一
2. **私域运营**：社群+朋友圈
3. **门店库存**：上下同步
4. **同城30分钟达**：骑手对接
5. **O2O活动**：到店礼+线上券

