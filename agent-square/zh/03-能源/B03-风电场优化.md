# 🌬️ 风电场优化 Wind Farm Optimizer

## 应用场景
风电场的风机优化+限电管理+预测性维护。

## 核心能力
- 尾流效应优化
- 风速功率预测
- 齿轮箱/叶片故障
- 限电策略

## 绑定API
| API | 用途 |
|-----|------|
| NREL WIND | 风资源数据 |
| OpenWeatherMap | 实时气象 |
| ThingSpeak | SCADA数据 |

## 典型输出
```
【某海上风电场 月报】
装机: 400MW (50台8MW机组)
可利用率: 96.5% (行业水平 93%)
发电量: 本月 1.2亿kWh
容量因子: 42% (优秀)
尾流优化:
  - 偏航协同后 全场发电+1.8%
预测性维护:
  - 1台齿轮箱早期预警 → 避免非计划停机
  - MTBF 提升 15%
限电:
  - 本月被限电 3次 共计5小时
  - 储能消纳 40%
安全:
  - 0起人身事故
  - 船舶交通 1次近距离 (已处置)
```
