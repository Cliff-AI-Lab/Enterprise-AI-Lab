# 💰 储能套利管家 Battery Storage Arbitrage Agent

## 应用场景
电力现货市场波动下，储能电站的实时充放电决策，最大化套利收益。

## Agent组合
```
[电力负荷预测员] → 电价预测
[新能源投研员] → 电芯健康度
[股票分析师] → 期货信号辅助
```

## 绑定API
| API | 用途 |
|-----|------|
| EIA Hourly | 现货电价(美国) |
| OpenWeatherMap | 新能源出力 |
| Alpha Vantage | 能源期货 |

## 核心工作流
1. **电价预测**：未来24小时
2. **SOC规划**：DP动态规划
3. **充电触发**：低价时段
4. **放电触发**：高价时段
5. **电芯保护**：DoD/温度
