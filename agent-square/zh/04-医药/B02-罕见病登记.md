# 🧩 罕见病登记 Rare Disease Registry

## 应用场景
政府/科研/药企的罕见病患者登记研究+药物开发+保险支付。

## 核心能力
- 真实世界登记
- 自然病史研究
- 药物试验招募
- 保障政策研究

## 绑定API
| API | 用途 |
|-----|------|
| ClinicalTrials.gov | 临床试验匹配 |
| OpenFDA Orphan | 罕见病药物 |
| PubMed | 文献 |
