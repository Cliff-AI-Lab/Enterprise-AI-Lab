# Enterprise AI Lab

Open-source AI solutions for enterprise scenarios across multiple industries. Each sector contains production-ready applications with modern tech stacks and AI-native architectures.

## Agri-Livestock

| Project | Description | Tech Stack |
|---------|-------------|-----------|
| [**Market Assistant**](agri-livestock/market-assistant/) | AI-powered commodity price forecasting & procurement decision platform for feed ingredients (soybean meal, corn, rapeseed meal, fishmeal, soybean oil) | React 19, Tailwind v4, ECharts, Framer Motion |

---

## Getting Started

Each project is self-contained with its own README, dependencies, and setup instructions. Navigate to the project directory and follow its Quick Start guide.

```bash
# Example: run the Market Assistant
cd agri-livestock/market-assistant
npm install
npm run dev
```
